NOTE : Input format should be x:yy:zz OR yy:zz where x is hours yy is minutes and zz is seconds.
Minimum Limit : 5:00, Maximum Limit : 2:00:00

  • Decide on the task to be done.
  • Set the session and break durations.Click on start button.
  • While you work the focus tree grows which yields you coins.
    If you cancel the session before it's done the tree will die and you won't get any rewards.
  • After session is done, take a break and start again with your task.
  • For every session you'll earn
    1. 1 coin (5-9 minutes)
    2. 5 coins (10-29 minutes).
    3. 20 coins (30 minutes).

You can view a graph of the session durations by clicking on stats button on the top right corner.
